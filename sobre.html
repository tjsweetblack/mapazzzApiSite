Sobre o MapaZZZ

O MapaZZZ é uma ferramenta desenvolvida para combater a malária através da tecnologia e da participação comunitária.

Documentação da API

A API do MapaZZZ permite acesso a dados em tempo real sobre zonas de risco e reportagens. Veja como usar:

Endpoint: /api/reportagens/zonas

Descrição: Retorna uma lista de zonas de risco identificadas.

Método: GET

Exemplo: fetch('https://mapazzz-api.vercel.app/api/reportagens/zonas')

Resposta: JSON com local, risco, latitude e longitude.

Endpoint: /api/reportagens

Descrição: Retorna reportagens enviadas pela comunidade (em desenvolvimento).

Método: GET

Exemplo: fetch('https://mapazzz-api.vercel.app/api/reportagens')

Para mais detalhes, entre em contato com a equipe do MapaZZZ.
